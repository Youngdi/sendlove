• { 我們的社會目的是… }

    從聘僱身心受限孩子成立烘焙坊，到現在為台灣友善農業、糧食自主而努力，喜願對環境、對人皆真誠以待。並於2014年開始，進口機器協助磨粉，降低農民磨豆的技術門檻，使粉能加工製成麵包、包子，活絡社區經濟發展。

    { 你們的消費支持了… }

    喜願麵包提供11位身心受限夥伴工作機會；喜願小麥七年來，與185位農友友善契作，栽種面積約485公頃小麥、45公頃大豆、4.5公頃蕎麥、3公頃芝麻，並以比進口高出30%-100%倍的價格收購(依照品項不同而有不同)。
